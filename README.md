 Expense Tracker
React-приложение, чтобы считать расходы и доходы.  

Доход — положительное число, расход — отрицательное. Enter отправляет форму.

Каждую транзакцию можно удалить отдельно. Есть кнопка «Clear all».
 
Автоматически считает:  
- Общий баланс  
- Сколько заработали (income)  
- Сколько потратили (expense)

Доход — зелёная полоса справа, расход — красная. Быстро видно, что есть что.

 
После перезагрузки страницы всё остаётся. Использовал localStorage.
